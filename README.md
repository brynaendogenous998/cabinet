## 🛠️ Troubleshooting

### The app does not open

- Check that the install finished.
- Right-click the app and choose **Run as administrator**.
- Restart your PC and try again.

### Windows blocks the app

- Look for a **More info** button.
- Choose **Run anyway** if you trust the source.
- Make sure you downloaded it from the page above.

### AI search is slow

- Check your internet connection.
- Close other apps that use bandwidth.
- Try again after a short wait.

### Text or layout looks too small

- Change Windows display scaling.
- Maximize the app window.
- Use the built-in zoom if the app provides it.
